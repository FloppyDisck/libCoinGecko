# libCoinGecko
A c++ API wrapper for Coin Gecko

## Setup
You must first install some libraries
Install njohmann's json
```
git clone https://github.com/nlohmann/json.git
cd json
mkdir build
cd build
cmake ..
sudo make install -j4
```

Install cpp-httplib
```
https://github.com/yhirose/cpp-httplib.git
cd cpp-httplib
mkdir build
cd build
cmake ..
sudo make install -j4
```
